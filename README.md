# 🐾 Pet Breed AI

An intelligent Flutter-based mobile application that helps users **identify pet breeds, interact with an AI chatbot, and find the perfect pet match** using smart recommendations.

---

## 📱 Features

### 🤖 AI Chatbot

* Ask anything about pet breeds
* Powered by Gemini API
* Provides real-time intelligent responses

### 🧠 Smart Match System

* Recommends pets based on:

  * Lifestyle
  * Living space
  * Budget
  * Activity level

### 📚 Breed Encyclopedia

* Detailed information about **35+ pet breeds**
* Covers dogs, cats, birds, and more

### 💡 Facts Section

* Learn interesting and fun pet facts

## 🚀 Getting Started

### Prerequisites

* Flutter SDK installed
* Android Studio / VS Code
* Emulator or Physical Device

### Installation

```bash
git clone https://github.com/your-username/pet-breed-ai.git
cd pet-breed-ai
flutter pub get
flutter run
```

---

## 🔐 API Setup (IMPORTANT)

This project uses Gemini API.

👉 Add your API key in:

```
lib/gemini_service.dart
```

Replace:

```dart
static const String apiKey = "YOUR_API_KEY";
```

⚠️ Do NOT expose your API key publicly.
